# 📦 COMO VERIFICAR BACKUPS NO SUPABASE

## 🎯 PASSO A PASSO

### **1. Acesse o Dashboard do Supabase**
- URL: https://app.supabase.com
- Faça login na sua conta

### **2. Selecione seu Projeto**
- Clique no projeto que contém o banco de dados

### **3. Vá para Settings**
- No menu lateral esquerdo, clique em **Settings** (Configurações)
- Ou acesse diretamente: https://app.supabase.com/project/[SEU_PROJECT_ID]/settings/database

### **4. Acesse Database → Backups**
- No menu Settings, clique em **Database**
- Role a página até encontrar a seção **Backups**
- Você verá uma lista de backups disponíveis

### **5. Verifique os Backups Disponíveis**
- Os backups são criados automaticamente pelo Supabase
- Procure por backups anteriores à data/hora da exclusão
- Clique em um backup para ver detalhes

### **6. Restaurar um Backup (SE NECESSÁRIO)**
⚠️ **CUIDADO:** Restaurar um backup vai **sobrescrever** todos os dados atuais!

- Clique no botão **Restore** ou **Download**
- Confirme a restauração
- Aguarde o processo completar

## 🔍 ALTERNATIVAS SE NÃO HOUVER BACKUP

### **Opção 1: Recriar o Jogo Manualmente**
Se você tiver:
- Título do jogo
- Lista de participantes (nome, número, WhatsApp)

Posso ajudar a recriar o jogo e adicionar os participantes.

### **Opção 2: Verificar Outras Fontes**
- Exportações CSV/Excel anteriores
- Screenshots da lista de participantes
- Histórico de notificações enviadas
- Logs do sistema em outro lugar

### **Opção 3: Contatar Suporte do Supabase**
Se você tiver plano pago, o suporte pode ter backups mais antigos:
- Email: support@supabase.com
- Discord: https://discord.supabase.com

---

## 📋 INFORMAÇÕES QUE PRECISO PARA RECRIAR

Se você não tiver backup, mas tiver estas informações, posso recriar:

1. **Título do jogo**
2. **Descrição do jogo**
3. **Lista de participantes:**
   - Nome
   - Número da sorte
   - WhatsApp (opcional)

# 🔍 Ver Registros DNS na Vercel

## 📊 O que fazer agora:

### **Passo 1: Clicar no link azul**

Na tela que você está vendo, há um link azul que diz:

**"Veja os registros DNS e mais para zkoficial.com.br →"**

**Clique nesse link!** Ele vai mostrar os registros DNS que você precisa configurar no seu provedor de domínio.

---

## 📋 O que você vai ver:

A Vercel vai mostrar algo como:

### **Opção A: CNAME**
```
Tipo: CNAME
Nome: www
Valor: cname.vercel-dns.com
```

### **Opção B: Registros A**
```
Tipo: A
Nome: @
Valor: 76.76.21.21

Tipo: A
Nome: www
Valor: 76.76.21.21
```

### **Opção C: Outros registros**
A Vercel pode pedir outros tipos de registros dependendo da configuração.

---

## 🔧 Próximos Passos:

1. ✅ **Clique no link azul** "Veja os registros DNS e mais para zkoficial.com.br →"
2. 📸 **Tire uma captura de tela** dos registros DNS que aparecerem
3. 📝 **Anote os registros** que a Vercel está pedindo
4. 🔧 **Configure esses registros** no seu provedor de domínio
5. ❌ **Remova os registros antigos** (que apontam para 162.240.81.81)
